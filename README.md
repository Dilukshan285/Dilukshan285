### 🔬 [FedMed — Federated Multi-Modal Medical AI](https://github.com/Dilukshan285) · `RESEARCH IN PROGRESS` · 2025–Present
Privacy-preserving federated learning system diagnosing **10 diseases** (Sepsis, Heart Failure, Pneumonia, Diabetes, COPD & more) across 2 hospitals using MIMIC-IV data — without sharing raw patient records.
- Custom **DW-FedAvg** algorithm weights each hospital's influence proportionally to its positive-sample counts
- **5-channel multi-modal architecture** fuses 17 universal labs, 8 specialist labs, chest X-rays (ResNet-50), and clinical notes (BioClinicalBERT)
- Achieves **0.89 AUROC**; externally validated on eICU dataset

`Python` `PyTorch` `MIMIC-IV (BigQuery)` `DW-FedAvg` `BioClinicalBERT` `ResNet-50` `CUDA` `NumPy` `Pandas`
